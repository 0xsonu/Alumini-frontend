# college Frontend Monorepo

## You have to only use Yarn don't use npm

Install Dependencies:

```sh
$ yarn install
```

If you want to install package from npm:

```sh
$ yarn workspace @frontend/web add <package name>
```

# Next.js Web

Run Web Next.js:

```sh
$ yarn web
```

Build Web Next.js:

```sh
$ yarn web:build
```
